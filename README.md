# 3D Heat Equation Solver (JAX): FDM vs PINN

Репозиторий содержит реализацию и сравнительный анализ двух подходов к решению нестационарного трехмерного уравнения теплопроводности с использованием фреймворка JAX.

## 📌 О проекте
Сравниваются два метода на задаче с известным аналитическим решением (условия Дирихле):
1. **FDM (Finite Difference Method):** Явная схема Эйлера, векторизованная с помощью `jax.lax.scan`.
2. **PINN (Physics-Informed Neural Network):** Многослойный перцептрон, минимизирующий невязку PDE в точках коллокации.

## 🚀 Установка и запуск

**Требования:**
- Python >= 3.9
- JAX >= 0.4.0

```bash
# Установка зависимостей
pip install -r requirements_pinn.txt

# Запуск демонстрации (сравнение с аналитическим решением)
bash run_demo.sh
```

## 📂 Структура репозитория
- `configs/` — YAML конфигурации для запуска экспериментов.
- `src/` — Исходный код солверов (`fdm_solver.py`, `pinn_solver.py`) и утилиты валидации.
- `experiments/` — Скрипты запуска.
- `results/` — Сохраненные графики сходимости и истории лоссов.
