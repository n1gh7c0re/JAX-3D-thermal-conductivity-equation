# [JAX-2-27] Уравнение теплопроводности 3D


## Background

Трехмерное моделирование теплопереноса необходимо для анализа терморегуляции электроники, процессов затвердевания материалов и климатического моделирования. 
Переход к 3D кратно увеличивает вычислительную сложность, делая актуальным применение физически-информированных нейросетей (PINNs) и GPU-ускоренных сеточных методов (JAX/CuPy) 
для преодоления «проклятия размерности».

## Task
Реализовать на JAX алгоритм решения нестационарного уравнения теплопроводности в трехмерной области.

Необходимо провести сравнительный анализ двух подходов:
1. Классический сеточный метод (FDM/FEM) с реализацией векторизованных операций в JAX (jax.lax.scan для времени)
2. Предложить архитектуру Physics-Informed Neural Networks (PINN), где решение аппроксимируется нейросетью , минимизирующей невязку уравнения в точках коллокации
