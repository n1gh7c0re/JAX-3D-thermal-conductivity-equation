# FDM Solver 3D

**Реализация классического сеточного метода** решения нестационарного 3D уравнения теплопроводности  

$$
u_t = α (u_{xx} + u_{yy} + u_{zz})
$$

с нулевыми граничными условиями Дирихле.


### Назначение кода
- Решает тестовую задачу аналитическим FDM-методом (явная схема Эйлера + центральные разности 2-го порядка).
- Использует **JAX** с `jax.lax.scan` и `jit` для максимальной производительности.
- Проверяет корректность реализации через сравнение с точным аналитическим решением.
- Выдаёт таблицу сходимости по ошибкам L₂ и L∞ — именно то, что требуется в отчёте курсового проекта.

### Описание кода

**fdm_solver.py** — основной солвер:
- `create_initial_condition` — создаёт начальное условие `sin(πx/Lx)·sin(πy/Ly)·sin(πz/Lz)`
- `laplacian_3d` — вычисляет оператор Лапласа во внутренних узлах
- `euler_step` — выполняет один шаг явной схемы Эйлера, границы автоматически остаются нулевыми
- `solve_fdm_3d` — главный цикл по времени через `lax.scan`, проверка устойчивости (CFL)

**test_fdm.py** — тесты и валидация:
- Проверяет, что граничные условия строго равны нулю
- Запускает решение на сетках 11³, 21³, 31³, 41³
- Сравнивает численное решение с аналитическим
- Выводит таблицу ошибок $L_2$ / $L_∞$ и шагов по времени
